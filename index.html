Course Refreshers

Six self-contained refresher courses on the applied-economics / data-science stack, each built as slides + written notes + runnable drills. Every course threads the same running example — a ride-sharing platform (Uber/Lyft) — through its modules, so switching topics doesn't mean switching domains. Built for interview prep; doubles as a working portfolio.
